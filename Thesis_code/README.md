# Code for thesis: Optimal growth strategies in microbial resource allocation models

This repository contains the Python code used to generate the numerical results and figures in my thesis.

## Contents

- `scripts/step_model_optimization.py`: simulations for the step-function model
- `scripts/MM_model_optimization.py`: simulations for the Michaelis-Menten model
- `scripts/phase_transition_plot.py`: code for phase-transition plots

## Requirements

Python version used: 3.11

```bash
pip install -r requirements.txt
